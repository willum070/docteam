Get Started with Codalab Worksheets

Using CodaLab worksheets is fun and easy. In this tutorial, you will use the CodaLab command-line client to run some experimental code against a simple dataset.

  1. Fork and clone the CodaLab repo.
  2. Install the command-line client: 
    pip install codalab-cli
  3. Evaluate your own data:
    4. 
cl work bundles.codalab.org:2800
cl upload 
cl run boosted-decision-trees
