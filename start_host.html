Host a Competition

Have a problem related to machine learning, medical imaging, or other field involving advanced computation? Organize a competition and call forth the talents of community members from far and wide!

Competitions are composed of a set of files called a 'bundle' which is stored on the CodaLab server, and competition data which is stored remotely. The bundle contains:

  • A configuration file (competition.yaml) that defines the competition and links to competition data.
  • A set of HTML files that contain information and instructions for participants.
  • An evaluation program that is used to score submissions.

To Host a Competition

  1. Create a competition bundle. For more details, see Building a Competition Bundle.
  2. Open a web browser and go to My CodaLab. If you do not have an account you will be prompted to create one.
  3. Click the Competitions I'm Running tab.
  4. Click Create Competition. You will be prompted to upload the bundle you created.
  5. See Competition Roadmap for a complete walkthrough.
