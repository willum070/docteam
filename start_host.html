Host a Competition

Have a problem related to machine learning, medical imaging, or other field involving advanced computation? Organize a competition and call forth the talents of community members from far and wide!

Competitions are composed of a bundle, which is a set of files. These files include files that link to the competition data, HTML pages that describe how the competition should work, and some type of evaluation program. The file competition.yaml is the configuration file that ties it all together.

To Host a Competition

  1. Create a competition bundle. For more details, see Competition Roadmap.
  2. Open a web browser and go to My CodaLab. If you do not have an account you will be prompted to create one.
  3. Click the Competitions I'm Running tab.
  4. Click Create Competition. You will be prompted to upload the bundle you created.
  5. See Running a Competition to learn how to manage your competition.
